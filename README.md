# 📊 Mobile Sales Analytics Dashboard | Power BI

An interactive **Power BI dashboard** built to analyze mobile sales performance, monitor key business KPIs, and provide meaningful business insights through dynamic visualizations.

---

## 📌 Overview

This project provides a comprehensive analysis of mobile sales data using Power BI. It enables users to monitor sales trends, compare year-over-year performance, and explore data through interactive reports and filters.

---

## 🚀 Features

- 📈 Interactive Dashboard
- 📅 Month-to-Date (MTD) Sales Analysis
- 📊 Same Period Last Year (SPLY) Comparison
- 🗺️ City-wise Sales Analysis
- 📱 Mobile Model Performance
- 🏷️ Brand-wise Sales Analysis
- 💳 Payment Method Distribution
- 📅 Monthly & Daily Sales Trends
- 🎯 Dynamic Filters & Slicers

---

## 📊 Key KPIs

- **Total Sales**
- **Total Quantity Sold**
- **Total Transactions**
- **Average Selling Price**

---

## 🛠️ Technologies Used

- Power BI
- DAX
- Power Query
- Data Modeling
- Data Visualization
